# AWS SageMaker란?

## 개념 정의
AWS SageMaker는 **클라우드에서 머신러닝(ML) 모델을 쉽고 빠르게 구축, 학습, 배포, 운영할 수 있게 해주는 완전 관리형 서비스**다. 데이터 전처리, 모델 개발, 자동 학습/튜닝, 대규모 배포 및 모니터링까지 ML의 모든 단계(integrated ML workflow)를 한 곳에서 지원한다.

쉽게 말해 “데이터 과학자와 개발자가 복잡한 인프라 신경 쓰지 않고, 바로 ML/AI 모델을 설계~학습~서빙까지 끝낼 수 있는 AWS의 올인원 AI 플랫폼”이다.

## 주요 특징 및 기능

**1. 통합 개발 환경(노트북)**
- SageMaker Studio: 웹 기반 IDE, 코드 작성·실행·시각화·협업 지원
- Jupyter 노트북 내장, 데이터 준비~모델 개발~실험 관리까지 한 곳에서 처리

**2. 데이터 전처리 및 탐색**
- S3, Redshift, RDS, Glue 등 다양한 데이터 소스 연동 및 자동 Pipeline 구축
- 데이터세트 전처리, 탐색(EDA) 도구 내장

**3. 강력한 모델 학습/튜닝**
- 빌트인 알고리즘(약 20여 개) + 오픈소스 프레임워크(TensorFlow, PyTorch, SKLearn, XGBoost 등) 모두 지원
- 대규모 분산 학습 환경(자동 스케일 및 GPU/TPU 지원)
- 자동 하이퍼파라미터 튜닝(AutoML, Hyperparameter Optimization)
- 스팟 인스턴스/분산 학습 지원으로 비용 절감

**4. 자동화 MLOps/실험 관리**
- 모델 버전 관리, 실험(Experiment), 파이프라인(ML Pipeline), 재현성(재현 가능한 워크플로) 지원
- 데이터~모델~서빙~모니터링 자동화(MLOps 실무 구조)

**5. 손쉬운 모델 배포/서빙**
- 한 줄 명령으로 대규모 확장(Endpoint) 배포
- 실시간 추론 REST API, Batch Transform(대량 추론) 모두 지원
- 모델 A/B 테스트, Blue/Green Deployment, Canary Release 지원

**6. 모델 모니터링 및 관리**
- 실시간 성능 모니터링, 드리프트 감지, 자동 알림
- 데이터 품질 검사(데이터 변칙, 예측 오류 등)

**7. 강화학습 및 딥러닝 지원**
- 시뮬레이션 기반 RL(강화학습) 플랫폼
- 최신 딥러닝 프레임워크 및 커스텀 컨테이너 지원

**8. 생성형 AI 통합**
- Bedrock, Titan 등 AWS 생성형 AI 모델 연동
- 외부 LLM, HuggingFace 등 직접 통합 가능

## 실무 적용 사례

- 기업/서비스의 자동 분류, 추천, 예측, 이상탐지, 챗봇, AI 기반 데이터 분석
- 핵심 BI·업무 프로세스 자동화, 센서·IoT·메타버스·이미지/음성/텍스트 처리
- MLOps 도입 및 전체 ML 워크플로의 자동화·최적화

## 비용 구조

- 사용 리소스(노트북, 학습 인스턴스, 배포 Endpoints, 스토리지, Pipeline 단계 등)별 Pay-as-you-go 과금
- 스팟 인스턴스, Batch Transform 등 비용 최적화 기능 제공

## 보안 및 운영

- IAM 역할, VPC, 암호화, 네트워크·스토리지·Endpoint 접근 제어 지원
- Model Registry, 실험 로그, 데이터 버전 관리로 규정 준수 및 감사활동 용이

## 요약

SageMaker는 “클라우드에서 ML을 직접, 대규모, 자동화·운영까지 모두 해결”하는 AWS의 대표 머신러닝/AI 서비스로, 데이터 과학·개발자·엔터프라이즈 모두 생산성을 극대화할 수 있는 필수 플랫폼이다.

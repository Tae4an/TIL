# Amazon Rekognition란?

## 개념 정의

Amazon Rekognition은 이미지·영상에서 객체, 사람, 텍스트, 장면, 활동, 부적절한 콘텐츠 등을 자동으로 인식·분석해 주는 완전관리형 컴퓨터 비전 서비스다. 딥러닝 모델이 이미 내장돼 있어, ML 지식 없이도 API 호출만으로 얼굴 인식, 콘텐츠 필터링, 텍스트 추출 같은 기능을 애플리케이션에 쉽게 추가할 수 있다.

쉽게 말해, “사진·동영상만 넘기면 사람/사물/텍스트/유해 콘텐츠를 자동으로 찾아주는 이미지·영상 분석 서비스”다.

## 핵심 기능

- **객체·장면 인식**  
  사람, 차량, 동물, 음식, 실내/실외, 바다/도시 등 다양한 객체와 장면을 라벨로 감지한다.

- **텍스트 감지 (OCR)**  
  이미지·영상 속 인쇄 및 일부 수기 텍스트를 감지·추출하며, 간판·배너·제품 패키지 등 기울어진 텍스트도 인식할 수 있다.

- **콘텐츠 모더레이션**  
  노출·성인물·폭력 등 안전하지 않은 콘텐츠를 감지하여 서비스에서 자동 필터링하는 데 사용된다.

- **얼굴 탐지·분석**  
  얼굴 위치, 표정(감정), 나이대 추정, 성별, 안경·수염 등 속성을 분석한다.

- **얼굴 인식·비교**  
  두 이미지 간 얼굴이 동일인인지 비교하거나, 사전에 등록한 얼굴 컬렉션에서 유사 인물을 검색한다.

- **유명 인사 인식**  
  이미지·영상에서 유명 인사를 인식해 태깅한다.

- **비디오 분석**  
  저장된/스트리밍 비디오에서 객체·장면·활동 감지, 사람 이동 경로 추적, 세그먼트(광고 구간, 엔딩 크레딧 등) 감지 등을 수행한다.

- **Custom Labels**  
  “헬멧 착용 여부”, “특정 부품/제품”, “브랜드 로고”처럼 비즈니스 도메인 특화 객체를 스스로 라벨링한 이미지로 학습시켜, 커스텀 모델을 자동 생성할 수 있다.

## 주요 활용 사례

- **유해 콘텐츠 필터링**  
  UGC(사용자 생성 이미지/영상)에서 노출·폭력 이미지 자동 차단.

- **본인 인증·접근 제어**  
  회원 가입/로그인 시 얼굴 비교로 신분증 사진과 사용자의 셀피를 매칭해 본인 여부 확인.

- **미디어·엔터테인먼트**  
  방송/영상 콘텐츠에서 장면 전환, 광고 구간, 출연자, 경기 하이라이트 구간 등을 자동 탐지해 편집·검색·추천에 활용.

- **소매/마케팅**  
  매장 CCTV 영상에서 방문객 수, 체류 시간, 동선 등을 분석하거나, 상품 진열 상태를 자동 점검.

- **안전·작업장 모니터링**  
  공장에서 작업자가 헬멧·조끼·장갑 등 보호구(PPE)를 착용했는지 감지.

- **문서·간판 텍스트 추출**  
  영수증, 전광판, 도로 표지판 등의 텍스트를 추출해 검색/분류/분석에 활용.

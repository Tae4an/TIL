# Amazon QuickSight란?

## 개념 정의
Amazon QuickSight는 AWS에서 제공하는 클라우드 네이티브 비즈니스 인텔리전스(BI) 서비스다. 서버리스 아키텍처 위에서 동작하며, 대시보드·리포트·시각화·애드혹 분석을 통해 다양한 데이터에서 인사이트를 빠르게 도출하고 공유할 수 있도록 돕는다.

쉽게 말해 “AWS 위에서 돌아가는 서버 관리 필요 없는 BI/대시보드 플랫폼”으로, 브라우저만으로 데이터 연결→시각화→공유까지 한 번에 처리할 수 있게 해주는 서비스다.

## 핵심 특징

- 서버리스·자동 확장  
  - 인프라 프로비저닝 없이 완전 관리형으로 제공되며, 수십·수백만 사용자까지 자동 확장된다.
  - 용량 계획이나 서버 관리 없이 대시보드 조회 시점에만 리소스를 사용하도록 설계되어 있다.

- 다양한 데이터 소스 연동  
  - Amazon S3, Redshift, RDS, Athena, Aurora 등 AWS 데이터 서비스는 물론, 온프레미스 DB, Salesforce, Excel/CSV 파일 등 외부 소스도 지원한다.
  - 라이브 쿼리 또는 SPICE(in-memory 엔진)에 데이터 적재 후 고속 분석이 가능하다.

- SPICE 인메모리 엔진  
  - SPICE(Super-fast, Parallel, In-memory Calculation Engine)는 데이터를 메모리에 복제해 고속 쿼리와 높은 동시성을 제공한다.
  - 다수 사용자가 동시에 대시보드를 조회해도 성능 저하 없이 응답할 수 있도록 설계되어 있다.[2]

- AI/ML 기반 분석 기능  
  - 이상 탐지, 예측(포캐스팅), 자동 인사이트 등 ML 기능을 내장해 비전문가도 쉽게 고급 분석을 수행할 수 있다.
  - QuickSight Q(자연어 질의)를 통해 사용자가 평문으로 질문하면 대시보드/차트 형태의 답을 제공한다.

- 임베디드 분석 및 API  
  - 애플리케이션·포털·SaaS에 대시보드와 리포트를 임베딩할 수 있으며, SSO 및 API를 통해 사용자 관리·배포 자동화가 가능하다.
  - 조직 내부용, 고객용 모두 커스텀 브랜드의 분석 화면을 쉽게 제공할 수 있다.

- 보안·거버넌스  
  - IAM 연동, 행(row)·열(column) 수준 보안, 암호화, CloudTrail 감사 등 엔터프라이즈급 보안 기능을 제공한다.
  - 멀티 리전·다중 AZ, 내장 중복성으로 고가용성과 규정 준수를 지원한다.

## 주요 사용 사례

- 경영/운영 대시보드: 매출·트래픽·인프라 상태 등 KPI 모니터링 대시보드 구성.
- SaaS/제품 내 임베디드 분석: 고객 포털에 직접 대시보드를 넣어 리포팅 기능 제공.
- 데이터 분석 셀프 서비스: 비즈니스 유저가 직접 필터링·드릴다운·애드혹 분석 수행.
- 이상 탐지/예측: 판매 추세, 수요 예측, 이상 거래 탐지 등 ML 기능을 활용한 분석.

## 과금 모델(요약)

- 완전 관리형 서버리스라 인프라 비용 대신 사용자/세션 기반으로 과금되며, 리더 사용자에 대해 pay-per-session 모델을 지원해 많은 사용자에게 경제적으로 배포할 수 있다.
- SPICE 용량, 사용자 유형(Author/Reader), 쿼리 방식(라이브/임포트)에 따라 요금이 달라진다.

Amazon QuickSight는 “서버 관리 없는 클라우드 BI + AI 기반 인사이트 + 임베디드 분석”을 결합한 서비스로, AWS 위에서 데이터 시각화와 분석 환경을 빠르게 구축하고 확장하고 싶을 때 가장 먼저 고려할 수 있는 도구다.

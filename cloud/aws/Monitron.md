# AWS Monitron이란

## 1. 개념 요약

Amazon Monitron은 **공장·플랜트 등 산업 현장의 회전 설비(모터·펌프·팬 등)에 무선 센서를 부착해 진동·온도 데이터를 수집하고, 기계 학습으로 이상 징후를 자동 감지하는 완전관리형 예측 유지보수 서비스**다. 센서·게이트웨이·클라우드 분석·모바일 앱까지 **엔드투엔드 패키지**로 제공되어, OT/IT 전문 인력 없이도 빠르게 도입할 수 있는 것이 가장 큰 특징이다.

***

## 2. 구성 요소

- **센서**
  - 설비(베어링, 모터 하우징 등)에 직접 부착하는 무선 센서로, 진동과 온도를 주기적으로 측정한다.
  - BLE(Bluetooth Low Energy)로 게이트웨이에 데이터를 전송하며, 별도 배선 없이 설치 가능하다.

- **게이트웨이**
  - 여러 센서에서 수집한 데이터를 Wi-Fi를 통해 AWS 클라우드로 업로드한다.
  - 하나의 게이트웨이에 여러 센서를 연결해 넓은 현장도 커버할 수 있다.

- **클라우드 분석 엔진**
  - AWS가 완전 관리하는 ML 모델과 ISO 20816 진동 기준으로 수집된 데이터를 분석한다.
  - 시간이 지날수록 설비별 패턴을 학습해 경보 정확도가 향상된다.

- **모바일 앱 / 웹 콘솔**
  - 센서·게이트웨이 등록·설치, 자산 상태 실시간 확인, 이상 알림 수신, 정비 이력 피드백 입력 등을 처리한다.

***

## 3. 동작 흐름

1. 설비에 **센서 부착** → 진동·온도 데이터 수집
2. **게이트웨이**가 데이터를 AWS로 전송
3. **클라우드 ML 모델**이 이상 징후(베어링 손상, 불균형, 과열 등) 분석
4. **모바일 앱**으로 관리자에게 알림 전송
5. 관리자가 고장 전에 **예방 정비** 계획 수립

***

## 4. 어디에 쓰이는가

- **제조 공장**: 컨베이어 벨트·컴프레서·펌프 등 핵심 설비의 베어링 손상·정렬 불량 조기 감지
- **물류센터**: 분류기·컨베이어 이상 감지로 라인 정지·출고 지연 최소화 (AWS 자체 물류센터 사례)
- **유틸리티·에너지**: 냉각 팬·순환 펌프 등 보조 설비 상태 모니터링
- **중소 규모 설비**: 고가 진단 장비 도입이 어렵던 보조 설비까지 감시 범위 확대

***

## 5. 예측 유지보수 방식 비교

| 항목 | 사후 유지보수 | 주기적 정기점검 | Monitron 예측 유지보수 |
|---|---|---|---|
| 정비 시점 | 고장 후 | 일정 주기 | 이상 징후 감지 시 |
| 다운타임 위험 | 매우 높음 | 중간 | 낮음 |
| 비용 효율 | 낮음 | 중간 | 높음 |
| 데이터 기반 | 없음 | 없음 | ML·센서 데이터 기반 |

***

## 6. 한 줄 정리

Amazon Monitron은 **무선 센서·게이트웨이·ML 분석·모바일 앱을 하나의 패키지로 제공해, 산업 설비의 이상 징후를 고장 전에 감지하고 예측 유지보수를 실현하는 AWS의 완전관리형 산업용 IoT·ML 서비스**다.

# AWS Security Hub란?

## 개념 정의
AWS Security Hub는 **AWS의 종합 클라우드 보안 관리 및 모니터링 서비스**다. AWS 환경(계정/리전/서비스 전체)의 다양한 보안 도구와 연동해, 보안 상태 진단·취약점 평가·알림 집계·컴플라이언스(규정 준수)·이상 탐지 등을 일원화된 대시보드와 API로 제공한다.

쉽게 말해 “AWS 계정/서비스 전반의 보안 현황을 한 눈에 파악하고, 취약점·위협 이벤트를 자동 수집·통합 관리·분석하는 클라우드 보안 관제 플랫폼”이다.

## 핵심 특징

**중앙집중식 보안 관리**
- 여러 AWS 계정·리전의 보안 이벤트, 경고, 취약점 정보 실시간 통합 집계
- 대시보드, 그래프, 리포트로 일목요연한 시각화

**다양한 보안 서비스·툴 연동**
- GuardDuty(위협 탐지), Inspector(취약점 분석), IAM Access Analyzer, Macie(민감 데이터 탐지), Firewall Manager 등 AWS 자체 서비스와 연동
- 외부 SIEM, 서드파티 보안 솔루션도 통합 가능(Cloud Security Alliance, Trend Micro 등)

**자동 규정 준수·표준 진단**
- CIS AWS Foundations Benchmark, PCI DSS 등 국제 보안 프레임워크 기반 자동 진단
- 계정/리소스별 컴플라이언스 스코어 산출 및 상세 리포트 제공

**통합 이벤트/알림**
- 모든 보안 이벤트를 Findings(파인딩, 경고) 객체로 통합
- EventBridge, CloudWatch, SNS 등과 연동해 알림·자동 대응/워크플로 트리거 가능

**자동화된 리미디에이션(대응)**
- Step Functions, Lambda 등과 연동해 취약점 발생 시 자체 대응 로직 자동 실행
- 지속적인 보안 상태 개선, 알림→대응의 자동화

**멀티 계정·리전 관리**
- 관리형/개별 계정, 모든 리전의 자원/이벤트 상태를 한 곳에서 확인
- AWS Organizations 연동시 중앙통제체계 운영 가능

## 주요 기능

- **대시보드 집계**: 전체 보안 현황, 규정 준수 점수, 취약점·이상 이벤트 시각화
- **자동 규정 준수 평가**: CIS, PCI, GDPR 등 기준에 따라 자동 점검/진단 리포트 생성
- **보안 파인딩 관리 및 자동 태깅**: 이벤트, 취약점, 조치 상태(해결됨/미해결 등) 자동 분류·상태 관리
- **서드파티 연동**: 타 보안 솔루션과 API/파인딩 동기화로 보안 생태계 확장
- **알림·자동 처리(리미디에이션)**: 즉각적 경고/알림과 대응 워크플로 구축(이벤트 중심 프로세스)

## 주요 활용 사례

- 클라우드 환경 확장 시 계정/리전별 보안 상태 집계·이상징후 탐지
- SOX, GDPR, HIPAA 등 컴플라이언스 준수 모니터링 및 자동 리포트 제출
- GuardDuty/Inspector 등 타 AWS 보안 서비스 결과 통합 분석
- 취약점·위협 발견 후 자동 대응(리미디에이션) 프로세스 코드화
- 보안 운영팀의 SIEM/SOC과 통합하여 실시간 경고·대응

## 워크플로/설정 예시

1. Security Hub 활성화(콘솔/CLI), AWS 계정·리전 등록
2. GuardDuty, Inspector 등 연동 서비스 활성화 및 파인딩 수집
3. 규정 준수 프레임워크(CIS/PCI 등) 정책 선택 및 평가 리포트 자동화
4. Alert/EventBridge/Lambda로 알림 및 자동 리미디에이션 워크플로 구축
5. 대시보드·리포트·API 등 운영에 맞는 정보 시각화·통합

## 비용 구조

- Security Hub 서비스 사용(Findings, 평가, 집계 등)에 따른 계정·리소스·리전별 월간 과금
- 서드파티 연동/외부 데이터 이동 비용 별도
- GuardDuty, Inspector 등 연계 서비스 자체 요금 추가

## 보안 및 운영 모범 사례

- 모든 계정·리전·서비스 통합 관리 기본 적용
- 취약점·이상 상황 발생 시 자동화된 대응 워크플로 구축(리미디에이션)
- 규정 준수 프레임워크 매월/주기적으로 평가, 리포트 자동화
- 알림·파인딩은 경고→조치→검증의 구조로 처리
- 외부 SIEM/SOC 연동으로 클라우드·온프레미스 통합 보안 관제 운용

AWS Security Hub는 클라우드 보안관제·취약점 관리·규정 준수·자동화 대응까지 올인원으로 제공하는 AWS 현행 보안 운영의 핵심 플랫폼이다.

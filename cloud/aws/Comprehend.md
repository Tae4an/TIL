# Amazon Comprehend란?

## 개념 정의

Amazon Comprehend는 텍스트에서 언어, 감정, 개체(사람·장소·조직), 핵심 키워드, 주제 등을 자동으로 추출해 주는 완전관리형 자연어 처리(NLP) 서비스다. 사전 학습된 머신러닝 모델을 API 형태로 제공하므로, 별도 모델 학습 없이 애플리케이션에 바로 텍스트 분석 기능을 넣을 수 있다.

쉽게 말해, “문장만 넘기면 ‘무슨 내용인지·어떤 감정인지·누가 언급됐는지·핵심이 뭔지’를 자동으로 뽑아주는 텍스트 분석 서비스”다.

## 핵심 기능

- **언어 감지**  
  텍스트가 어떤 언어(한글, 영어 등)인지 자동 식별.

- **감정 분석 (Sentiment Analysis)**  
  문장/문서가 긍정·부정·중립·혼합 중 어떤 감정인지 점수와 함께 반환.

- **개체 인식 (Entity Recognition)**  
  사람, 조직, 위치, 날짜, 제품명 등 의미 있는 엔티티를 찾아 유형별로 태깅.

- **키 프레이즈 추출 (Key Phrase Extraction)**  
  문장의 핵심 구문·키워드를 자동 추출해 요약·검색·태깅에 활용.

- **문서 분류 (Classification)**  
  뉴스/리뷰/문의글 등을 미리 정의한 카테고리(예: 기술 지원, 불만, 칭찬 등)로 자동 분류.

- **주제 모델링 (Topic Modeling)**  
  대량 문서 집합에서 자동으로 주제를 찾아 묶어주는 기능.

- **PII 탐지·마스킹**  
  개인정보(이름, 주민/여권 번호, 이메일, 주소 등)를 식별하고 마스킹/삭제에 활용.

- **Custom 엔티티/분류 (Comprehend Custom)**  
  도메인 특화 용어(예: 금융상품명, 의료용어)를 인식하거나, 조직별 카테고리 체계로 분류하도록 자체 데이터로 커스텀 모델 학습 가능.

## 주요 활용 사례

- **고객 피드백·리뷰 분석**  
  리뷰·콜센터 기록·설문 응답의 감정과 키워드를 분석해 고객 만족도, 불만 요인, 개선 포인트 파악.

- **소셜 미디어 모니터링**  
  트위터/커뮤니티 글에서 브랜드/제품 언급을 수집하고, 긍·부정 분위기와 주요 이슈 추출.

- **문의·티켓 자동 라우팅**  
  이메일·문의 글을 자동 분류해 “결제 문제/계정 문제/기능 문의” 등 담당 팀별로 자동 할당.

- **문서 검색 고도화 (Semantic Search)**  
  단순 키워드 검색이 아니라, 문서에서 추출한 개체·키워드·감정·주제를 인덱싱해 더 의미 기반의 검색 결과 제공.

- **법률·금융 문서 처리**  
  계약서·약관·보험 청구서 등에서 당사자, 날짜, 금액 등 핵심 엔티티를 추출하고, PII를 자동 마스킹.

- **규제·보안 대응**  
  문서·로그에서 개인정보를 자동 탐지·마스킹해 데이터 프라이버시 및 규정 준수에 활용.

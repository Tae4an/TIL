# AWS DeepLens란

## 1. 개념 요약

AWS DeepLens는 **딥러닝 모델을 디바이스 자체에서 직접 실행할 수 있는 개발자용 완전 프로그래밍 가능 무선 비디오 카메라**다. 단순한 카메라가 아니라 **하드웨어(카메라+컴퓨팅 보드)·소프트웨어·AWS 클라우드 연동이 패키지로 묶인 엣지 AI 개발·학습용 플랫폼**이며, 2017년 re:Invent에서 "세계 최초 딥러닝 지원 비디오 카메라"로 소개됐다.

***

## 2. 하드웨어 사양

- **카메라**: 4MP, 1080p 해상도, 내장 마이크
- **프로세서**: Intel Atom 프로세서 (듀얼 코어)
- **GPU**: Intel Gen9 GPU → 딥러닝 추론을 카메라 자체에서 실행
- **메모리/스토리지**: 8GB RAM, 16GB eMMC
- **OS**: Ubuntu 16.04
- **연결**: Wi-Fi, USB, Micro HDMI

***

## 3. 소프트웨어 스택

- **AWS Greengrass**: 클라우드에서 학습한 모델을 디바이스에 배포하고, 로컬에서 Lambda 함수를 실행하는 엣지 런타임
- **Apache MXNet**: 디바이스 최적화 버전이 기본 탑재된 딥러닝 프레임워크
- **Intel clDNN 라이브러리**: Intel GPU에서 딥러닝 추론을 가속하는 최적화 라이브러리
- **DeepLens Model Optimizer API**: SageMaker 등에서 학습한 모델을 Intel 칩에 맞게 자동 최적화

***

## 4. 동작 흐름

1. **SageMaker 또는 사전 학습 모델**로 컴퓨터 비전 모델 준비
2. **DeepLens 콘솔**에서 모델과 Lambda 함수를 묶어 프로젝트 생성
3. **Greengrass**가 모델을 디바이스에 자동 배포·최적화
4. **카메라 내 Intel GPU**에서 실시간 영상 추론 실행 (클라우드 전송 없이)
5. 결과를 AWS IoT·Lambda·S3·Rekognition 등으로 전송해 추가 처리

***

## 5. 제공 샘플 프로젝트

AWS가 사전 학습 모델과 코드를 함께 제공해 바로 실행할 수 있는 예제들이 있다.

- **얼굴 감지**: 화면 내 얼굴 위치·수 감지
- **객체 인식**: 100여 가지 사물 분류 (고양이, 자동차, 가방 등)
- **행동 인식**: 사람의 동작 분류 (앉기, 서기 등)
- **스타일 트랜스퍼**: 영상에 예술적 스타일 실시간 적용
- **핫도그 감지**: MXNet 기반 음식 분류기 (유명한 실습 예제)
- **수화 인식(ASLens)**: 미국 수화 문자 인식 후 Amazon Polly로 음성 출력

***

## 6. 어디에 쓰이는가

- **ML 교육·학습**: 딥러닝 입문 개발자가 컴퓨터 비전의 전체 사이클(학습→배포→추론)을 직접 실습
- **엣지 컴퓨터 비전 프로토타이핑**: 클라우드 의존 없이 카메라 자체에서 추론하는 엣지 AI 앱 빠른 개발
- **AWS AI 서비스 연동 실험**: Rekognition, Polly, Lambda, IoT와 조합한 파이프라인 구성 실습
- **해커톤·챌린지**: AWS가 주기적으로 DeepLens Challenge를 개최해 교육·접근성·의료 등 다양한 사회적 테마의 프로젝트를 공모

***

## 7. 현황 및 포지션

DeepLens는 **교육·개발자 경험용 제품**으로 출시되었으며, 이후 AWS는 실제 산업 현장용 엣지 AI로 **AWS Panorama**(카메라 어플라이언스·SDK), 설비 모니터링용 **AWS Monitron** 등 더 전문화된 엣지 AI 서비스로 진화시켰다. 신규 판매는 제한적이며, 지금은 학습 리소스·문서가 주로 참조용으로 남아 있다.

***

## 8. 유사 AWS 엣지 AI 서비스 비교

| 항목 | AWS DeepLens | AWS Panorama | AWS Monitron |
|---|---|---|---|
| 대상 | 개발자 학습용 | 기업 현장 카메라 분석 | 산업 설비 진동·온도 |
| 하드웨어 | 전용 카메라 디바이스 | 어플라이언스 + 기존 IP 카메라 | 무선 진동·온도 센서 |
| 주요 용도 | 컴퓨터 비전 실습 | 실시간 영상 분석 (품질·안전) | 예측 유지보수 |
| 성숙도 | 레거시 (신규 판매 축소) | 현재 주력 | 현재 주력 |

***

## 9. 한 줄 정리

AWS DeepLens는 **Intel GPU가 내장된 개발자용 딥러닝 비디오 카메라로, SageMaker·Greengrass·Lambda 등 AWS AI 스택과 긴밀히 연동해 컴퓨터 비전 모델의 학습부터 엣지 배포·추론까지 전 과정을 실습할 수 있는 엣지 AI 개발 플랫폼**이다.

# AWS CodeGuru란

## 1. 개념 요약

Amazon CodeGuru는 **머신러닝 기반 AI 코드 리뷰 및 런타임 성능 분석 서비스**다.  
AWS가 수천만 줄의 오픈소스 코드와 Amazon 내부 코드베이스로 훈련한 모델을 활용해, **개발자가 놓치기 쉬운 버그·보안 취약점·성능 병목을 자동으로 탐지하고 개선 권고를 제공하는 "AI 기반 코드 품질 자동화 도구"** 다.

***

## 2. 두 가지 핵심 구성 요소

- **CodeGuru Reviewer (정적 분석)**  
  - PR(Pull Request) 제출 시 코드를 자동 분석해 잠재적 버그·보안 취약점·리소스 누수·동시성 문제 등을 탐지.  
  - OWASP Top 10 보안 위험, AWS API 사용 모범 사례 위반, 하드코딩된 시크릿(비밀번호·API 키) 검출 기능 포함.  
  - Java, Python 지원. GitHub, Bitbucket, AWS CodeCommit 등 주요 리포지토리와 통합.  

- **CodeGuru Profiler (런타임 성능 분석)**  
  - 운영 중인 애플리케이션에서 실시간으로 CPU 사용률·지연 시간 데이터를 수집해 병목 메서드를 시각화.  
  - 비용이 많이 드는 코드 라인을 자동으로 찾아내 최적화 가이드 제공.  
  - EC2, ECS, EKS, Lambda 등 다양한 컴퓨팅 환경 지원.  

***

## 3. 특징

- **ML 기반 자동화**  
  - 단순 규칙 기반이 아닌 머신러닝 모델이 코드 패턴을 학습해 복잡한 잠재 결함까지 탐지한다.  

- **CI/CD 파이프라인 통합**  
  - CodePipeline, GitHub Actions, Jenkins 등과 통합해 코드 커밋 → 자동 리뷰 → 피드백 전달까지 자동화 가능.  

- **시크릿 탐지 (Secrets Detector)**  
  - 코드에 하드코딩된 비밀번호, API 키, AWS 자격증명 등을 자동 탐지하고 AWS Secrets Manager로 이관을 권고.  

- **이상 감지 자동화**  
  - Profiler가 앱 프로파일에서 평소와 다른 CPU 급증·지연 패턴을 자동 감지해 알림을 보낸다.  

***

## 4. 어디에 쓰이는가

- **자동화 코드 리뷰**  
  - PR마다 사람 리뷰어 없이도 자동으로 보안·품질 이슈를 검출해 개발 속도와 코드 품질을 동시에 높인다.  

- **보안 취약점 사전 차단**  
  - 배포 전 단계에서 OWASP 취약점·하드코딩된 시크릿을 자동 탐지해 보안 사고를 예방한다.  

- **성능 최적화**  
  - 운영 중 비용을 가장 많이 유발하는 메서드를 찾아내 Lambda 비용, EC2 CPU 낭비를 줄인다.  

- **DevSecOps 파이프라인**  
  - CI/CD 안에 보안 스캐닝과 품질 검사를 자동으로 내재화하는 DevSecOps 구현에 활용.

***

## 5. 유사 서비스와의 비교

| 항목 | AWS CodeGuru | SonarQube | Snyk |
|---|---|---|---|
| 분석 방식 | ML + 정적 분석 + 런타임 프로파일링 | 정적 분석(규칙 기반) | 보안 취약점 스캐닝 |
| 성능 분석 | 런타임 프로파일러 내장 | 없음 | 없음 |
| 보안 탐지 | OWASP, 시크릿 탐지 포함 | 규칙 기반 보안 탐지 | 의존성·코드 보안 중심 |
| AWS 통합 | 네이티브 | 별도 플러그인 | 별도 연동 필요 |
| 지원 언어 | Java, Python | Java, JS, Python 등 다양 | 다양한 언어 지원 |

***

## 6. 한 줄 정리

AWS CodeGuru는 **머신러닝으로 PR 코드를 자동 리뷰하고 운영 중 성능 병목까지 찾아주는 AI 기반 코드 품질·보안 자동화 도구**로, DevSecOps 파이프라인에서 코드가 배포되기 전 "AI 코드 리뷰어"로 동작한다.

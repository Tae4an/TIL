# Amazon Q란?

## 개념 정의
Amazon Q는 AWS에서 제공하는 **생성형 AI 기반의 기업용 챗봇 및 생산성 지원 비서 서비스**다. 사용자의 조직 데이터, 개발 리포지터리, AWS 문서, 내부 정책 등을 연결해 자연어 질의와 대화로 정보를 찾아주며, 개발/운영/보안 자동화, 분석 등 다양한 업무를 지원한다.

쉽게 말해 “AWS 클라우드와 기업 업무환경에 딱 맞춘 Copilot형 인공지능 비서”로, 내부 지식 조회, 코드 생성/리뷰, 리포트 작성, 워크플로 자동화 등 생산성 증대·AI 도입을 극대화하는 서비스다.

## 핵심 특징

**기업 전용 생성형 AI 기반**
- LLM(대형 언어 모델; Amazon Titan, Claude 등 다양한 AI 활용)
- 조직 내 자체 데이터/문서/코드/FAQ/정책 등 연결
- RAG(Retrieval-Augmented Generation)로 최신/신뢰성 높은 답변 제공

**업무별 AI 자동화**
- 개발자용: 코드 추천, 자동완성, 설명, 보안 이슈 탐지(IDE/CLI/AWS 콘솔 연동)
- 운영/DevOps: 인프라 문제 분석, 이벤트/알람 원인·처리 제안(Amazon Q DevOps)
- 보안/리스크: 알림 감지, 정책 설명, 규정 준수 가이드 지원

**대화형 인터페이스**
- Slack, Microsoft Teams, AWS Management Console, 전용 챗봇 인터페이스 등 다양한 환경 연동
- 회사 내 조직 구조·권한 체계 반영, 인증·접근 제어 적용

**내부 지식·문서 검색**
- 사내 위키, 정책, 백서, 데이터시트, 에버노트, S3 문서까지 통합 검색
- 질문→답변→즉시 링크·자료 제공 및 문맥 추적

**보안·개인화**
- 조직 데이터 사내 전용, Role/권한 기반 답변 필터링
- 사용자별 대화 이력, 선호 답변, 업무 맥락 학습

## 주요 활용 사례

- 개발자: 코드 리뷰, 실시간 코드 생성, 보안 패턴·모범 사례 안내
- 운영자: AWS 장애/오류 분석, 리소스 최적화 제안, 로그 분석
- 정책/보안팀: 내부 정책/규정 질의, 감사 자료 신속 확보
- 조직 전체: 사내 Q&A 지식, 프로젝트/문서 검색, FAQ/Helpdesk 자동화

## 아키텍처 및 연동

- RAG(질의 시 실시간 검색→생성), LLM엔진(Amazon Titan, Bedrock, OpenAI 등)
- 사내 S3·위키·데이터베이스·GitHub 등 외부/사내 시스템 연결(커넥터 플러그인)
- 내부 인증 연동(SSO, IAM 등) 및 접근 규칙 적용

## 비용 구조

- 요청/사용량·답변량·연동 데이터량 기준 과금(예상 사용량에 따라 티어 또는 요청 단위 과금)
- 사내 데이터 저장, 커넥터 연동, 관리 대상 확대에 따라 부가 요금 발생

## 보안 및 운영 모범 사례

- 접근 권한 최소화, 조직/부서/권한별 정보 분리
- 업무/개인별 데이터피드백·사용 패턴 정기 리뷰
- 보안 데이터 보호, 로그·이력·감사 기능 활성화
- 학습 데이터·모델 제어 및 사내만의 RAG 정책 구성

Amazon Q는 AWS 및 조직 내부 데이터 활용도를 극대화하고, 개발·운영·전사적 업무의 효율과 신뢰도를 혁신적으로 높여주는 차세대 AI 업무 파트너다.

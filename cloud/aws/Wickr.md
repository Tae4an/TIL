# AWS Wickr란

## 1. 개념 요약

AWS Wickr는 **종단 간 암호화(End-to-End Encryption)를 제공하는 엔터프라이즈용 보안 메신저·협업 서비스**다.  
조직과 정부·공공 기관이 **메시지, 음성·영상 통화, 파일·화면 공유를 높은 보안 수준으로 사용**할 수 있도록 설계된 보안 특화 커뮤니케이션 플랫폼이라고 보면 된다.

***

## 2. 특징

- **종단 간 암호화(E2EE)**  
  - 메시지·통화·파일·화면 공유 전체가 종단 간 암호화되며, 중간에서 데이터를 가로채도 내용을 볼 수 없다.  
  - 암·복호화 키는 사용자 단말(클라이언트)에만 존재하고, 서버는 암호화된 데이터만 저장한다.

- **보안·규제 대응**  
  - 제로 트러스트(Zero Trust) 철학, 강한 암호화 알고리즘, 엄격한 인증·접근 제어를 제공한다.  
  - 고보안 환경(정부·국방·규제 산업) 요구사항을 충족하도록 설계되었다.

- **정책·감사·관리 기능**  
  - 조직 정책에 따라 메시지 보존 기간, 자동 삭제, 포워딩 제한 등을 세밀하게 설정할 수 있다.  
  - 관리 콘솔에서 사용자·디바이스·보안 그룹·정책을 중앙에서 관리하고, 필요 시 원격 로그아웃·데이터 삭제도 가능하다.

- **엔터프라이즈 통합**  
  - AD/LDAP, SSO(OpenID Connect 등)와 연동해 기존 사내 계정으로 로그인 가능하다.  
  - Bot과 API를 통해 알림/자동화 시나리오, 다른 AWS 서비스와의 연계도 구성할 수 있다.

***

## 3. 어디에 쓰이는가

- **민감한 내부 커뮤니케이션**  
  - 경영진, 보안팀, 인수합병(M&A) 프로젝트 등 기밀도가 높은 대화 채널을 위한 메신저로 활용.

- **정부·국방·공공기관**  
  - 정책·작전·사고 대응 등 기밀 커뮤니케이션에 사용되는 고보안 메신저로 도입.

- **보안 사고 대응(Incident Response)**  
  - 내부망이 이미 침해되었을 수도 있는 상황에서, 별도의 안전한 채널로 CSIRT/SOC 팀이 상황 공유·지시를 주고받는 용도로 사용.

- **외부 파트너와의 안전한 협업**  
  - 컨설팅사, 공급사 등 외부 파트너와도 보안이 보장된 채널로 파일·메시지를 교환하고, 접근 권한을 세밀하게 제어.

***

## 4. 주요 기능 정리

| 기능 | 설명 |
|---|---|
| 보안 메시징 | 일대일·그룹 채팅, 메시지 자동 소멸(타이머), 메시지 복사·포워딩 제한 설정 등 |
| 음성·영상 통화 | 종단 간 암호화된 1:1 및 그룹 음성·영상 통화 |
| 파일·화면 공유 | 파일, 스크린, 위치 등을 암호화된 채널로 공유 |
| 오픈 액세스(네트워크 유연성) | 다양한 네트워크·방화벽 환경에서 안정적으로 통신하도록 설계 |
| 중앙 관리·감사 | 사용자·정책·로그 관리, 규제·감사 대응을 위한 기록 기능 제공 |

***

## 5. 한 줄 정리

AWS Wickr는 **메시지·통화·파일 공유를 종단 간 암호화와 강력한 정책·감사 기능으로 보호해, 기업·정부가 가장 민감한 커뮤니케이션을 안전하게 수행할 수 있게 해주는 엔터프라이즈급 보안 협업 플랫폼**이다.

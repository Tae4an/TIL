# Illumio Core란
## 1. 개념 요약
Illumio Core는 **데이터센터·클라우드·엔드포인트 전반에서 워크로드 단위의 제로 트러스트 마이크로세그멘테이션을 제공하는 Illumio의 핵심 보안 플랫폼**이다.  
쉽게 말해, **서버·VM·컨테이너에 에이전트(VEN)를 설치해 각 워크로드의 OS 기본 방화벽을 중앙에서 제어하고, 허용된 통신만 통과시켜 공격자의 횡방향 이동을 차단하는 "워크로드 중심 마이크로세그멘테이션 플랫폼"** 이다.

앞서 정리한 CloudSecure가 **클라우드 환경을 에이전트 없이** 보호하는 데 특화되어 있다면, Core는 **데이터센터·온프레미스·하이브리드 환경에서 에이전트 기반으로 OS 레벨까지 깊이 제어** 하는 것이 핵심 차이다.

***
## 2. 특징
- **에이전트 기반 워크로드 보호 (VEN)**  
  - 각 서버·VM에 VEN(Virtual Enforcement Node) 에이전트를 설치해, OS 내장 방화벽(iptables, Windows Firewall 등)을 Illumio가 중앙 정책에 따라 자동으로 제어한다.  
  - 인프라 환경(물리서버, VMware, AWS EC2, Azure VM 등)에 상관없이 동일하게 적용 가능하다.

- **레이블(Label) 기반 정책 모델**  
  - IP·서브넷·포트 기반이 아닌, **애플리케이션·역할·환경·위치 등의 레이블**을 기반으로 정책을 정의한다.  
  - IP가 바뀌어도 정책 변경이 필요 없고, 애플리케이션 맥락에 맞는 정책을 직관적으로 관리할 수 있다.

- **허용 목록(Allowlist) 모델**  
  - 기본적으로 모든 통신을 차단하고, **정책에 명시적으로 정의된 연결만 허용**하는 Allowlist 방식으로 동작한다.  
  - 정의되지 않은 트래픽은 기본 차단되어, 공격자가 새로운 경로를 열기 어렵다.

- **실시간 애플리케이션 의존성 가시성 (Illumio Map)**  
  - 전체 워크로드 간 트래픽 흐름을 실시간으로 시각화해, 어떤 서버가 어떤 서버와 통신하는지 한눈에 볼 수 있다.  
  - 실제 트래픽 흐름을 분석해 정책을 자동 추천해주는 기능도 제공한다.

- **중앙 정책 관리 서버 (PCE)**  
  - PCE(Policy Compute Engine)가 모든 VEN 에이전트와 통신하며, 정책 배포·상태 관리·로그 수집을 중앙에서 처리한다.  
  - SaaS 방식 또는 자체 관리형(On-Premises) 배포를 모두 지원한다.

***
## 3. 어디에 쓰이는가
- **데이터센터 내부 횡방향 이동 차단**  
  - 공격자가 서버 하나를 침해한 후 내부 시스템으로 확산하는 경로를 마이크로세그멘테이션 정책으로 차단.

- **랜섬웨어 확산 방지**  
  - 랜섬웨어 감염 시 해당 워크로드를 즉시 격리해 네트워크 전체로의 확산을 방지.

- **Active Directory, 핵심 인프라 보호**  
  - AD 서버·DB·핵심 시스템을 별도 세그먼트로 분리하고, 필요한 워크로드에서만 통신하도록 정책을 적용.

- **규제·컴플라이언스 준수**  
  - PCI DSS, HIPAA, NIST 제로 트러스트 아키텍처 등에서 요구하는 네트워크 세그멘테이션을 워크로드 단위로 구현.

- **AWS GuardDuty 등 보안 도구와 연동**  
  - GuardDuty, SIEM, SOAR와 연동해 위협 탐지 → 자동 격리 대응 파이프라인 구성 가능.

## 4. Illumio Core vs CloudSecure 비교
| 항목 | Illumio Core | Illumio CloudSecure |
|---|---|---|
| 주요 환경 | 데이터센터·온프레미스·하이브리드 | 퍼블릭 클라우드(AWS·Azure·GCP 중심) |
| 에이전트 여부 | VEN 에이전트 설치 필요 | 에이전트 없음 (Agentless) |
| 정책 시행 방식 | OS 내장 방화벽 직접 제어 | 클라우드 네이티브 보안 그룹(SG, NSG) 제어 |
| 가시성 수준 | 프로세스·포트 수준까지 상세 | 클라우드 워크로드·태그 기반 |
| 대상 워크로드 | 물리서버, VM, 컨테이너, 하이브리드 | 클라우드 VM, 컨테이너, 서버리스 |
| 공통점 | 레이블 기반 정책, 제로 트러스트 세그멘테이션, 횡방향 이동 차단 목적 ||

***
## 5. 한 줄 정리
Illumio Core는 **데이터센터·클라우드·하이브리드 환경에서 워크로드마다 VEN 에이전트를 통해 OS 레벨 방화벽을 중앙 제어하고, 레이블 기반 허용 목록 정책으로 공격자의 횡방향 이동을 차단하는 "워크로드 중심 제로 트러스트 마이크로세그멘테이션 플랫폼"** 이다.

# Illumio CloudSecure란
## 1. 개념 요약
Illumio CloudSecure는 **하이브리드·멀티 클라우드 환경에서 워크로드 간 제로 트러스트 마이크로세그멘테이션(Zero Trust Segmentation)을 제공하는 클라우드 네이티브 보안 솔루션**이다.  
즉, **클라우드 내부에서 침해가 발생했을 때 공격자가 다른 워크로드·서비스로 횡방향 이동(Lateral Movement)하는 것을 차단해, 피해 범위를 최소화하는 "클라우드용 마이크로세그멘테이션 플랫폼"** 이라고 보면 된다.

***
## 2. 특징
- **에이전트 없는(Agentless) 방식**  
  - 각 워크로드에 에이전트를 직접 설치하지 않고도 클라우드 네이티브 API와 연동해 트래픽 흐름 가시성과 세그멘테이션 정책을 적용한다.  
  - 대규모 클라우드 환경에서도 배포가 빠르고 운영 부담이 적다.

- **레이블 기반 정책 모델**  
  - IP·포트 같은 네트워크 주소 기반이 아니라, **비즈니스/애플리케이션 맥락에 맞는 레이블(태그)**을 기반으로 정책을 정의한다.  
  - 기존 클라우드 태그(AWS 태그, Azure 태그 등)를 Illumio 레이블에 자동 매핑해 정책을 구성할 수 있다.

- **클라우드 네이티브 보안 도구 활용**  
  - AWS Security Group, Azure NSG(Network Security Group) 같은 **클라우드 네이티브 보안 제어를 직접 프로그래밍**해 세그멘테이션 정책을 구현한다.  
  - 별도 보안 어플라이언스 없이 클라우드 기본 기능을 사용해 정책을 시행한다.

- **실시간 트래픽 가시성**  
  - 하이브리드·멀티 클라우드(AWS, Azure, GCP, 온프레미스) 전체에서 모든 워크로드 간 트래픽 흐름을 단일 뷰로 확인할 수 있다.  
  - 실시간 통신 패턴을 분석해 트래픽·태그·로그 맥락에 따라 정책을 자동으로 추천·조정한다.

- **침해 발생 시 자동 격리**  
  - 랜섬웨어·침해 탐지 시 손상된 세그먼트를 동적으로 격리해, 공격자의 내부 확산을 차단한다.  
  - 탐지 여부와 무관하게 세그멘테이션 정책이 항상 적용되어 있어 사전에도 이동을 막는다.

***
## 3. 어디에 쓰이는가
- **클라우드 내 횡방향 이동 차단**  
  - 공격자가 하나의 워크로드 침투 후 다른 서비스·DB·API로 확산하는 경로를 세그멘테이션으로 차단.

- **랜섬웨어 대응**  
  - 랜섬웨어 공격 탐지 즉시 감염 워크로드를 격리해 클라우드 전체로 확산되는 것을 방지.

- **클라우드 워크로드 마이그레이션 보안**  
  - 온프레미스에서 클라우드로 이전하는 동안 트래픽 흐름을 시각화하고, 마이그레이션 전후 일관된 보안 정책을 자동으로 유지.

- **멀티 클라우드·하이브리드 환경 통합 관리**  
  - AWS·Azure·GCP·데이터센터에 걸쳐 분산된 워크로드를 단일 플랫폼에서 가시화하고 일관된 제로 트러스트 정책을 적용.

- **규제·컴플라이언스 환경**  
  - PCI DSS, HIPAA, ISO 27001 같은 규정에서 요구하는 네트워크 세분화·접근 제어를 클라우드 환경에서 구현하는 데 활용.

***
## 4. 유사 솔루션과의 비교
| 항목 | Illumio CloudSecure | AWS Security Groups | 전통 방화벽(NGFW) |
|---|---|---|---|
| 적용 범위 | 멀티·하이브리드 클라우드 통합 | AWS 단일 클라우드 | 온프레미스·경계 보안 중심 |
| 정책 기반 | 레이블/애플리케이션 중심 | IP·포트 기반 | IP·포트·정책 기반 |
| 에이전트 | 불필요 (Agentless) | 해당 없음 | 해당 없음 |
| 가시성 | 전 환경 워크로드 간 트래픽 통합 뷰 | VPC 내 단위 확인 | 경계 트래픽 중심 |
| 횡방향 이동 차단 | 핵심 기능 | 직접 구성 필요 | 내부 트래픽 제어 어려움 |

***
## 5. 한 줄 정리
Illumio CloudSecure는 **에이전트 없이 멀티·하이브리드 클라우드 전체의 워크로드 트래픽을 실시간으로 가시화하고, 레이블 기반 제로 트러스트 마이크로세그멘테이션을 통해 침해 발생 시 공격자의 내부 확산을 자동 차단하는 클라우드 보안 플랫폼**이다.

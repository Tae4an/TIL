# Veeam이란?
## 1. 개념 요약
Veeam(빔)은 **가상화·클라우드·물리 환경을 아우르는 엔터프라이즈급 백업·복구·데이터 보호 솔루션**을 제공하는 글로벌 소프트웨어 기업이다.  
2006년 설립 이후 VMware·Hyper-V 가상화 특화 백업으로 성장했으며, 현재는 **AWS·Azure·GCP·SaaS까지 통합 지원하는 "하이브리드 클라우드 백업 플랫폼"** 으로 자리잡았다.

***
## 2. 특징
- **이미지 기반 백업**
  - VM 전체를 이미지로 백업해 OS, 앱, DB를 한 번에 보호
  - 스냅샷 기술로 서비스 중단 없이 백업 수행

- **Instant Recovery (즉시 복구)**
  - 백업본에서 VM을 즉시 기동 → 장애 시 수 분 내 서비스 재개
  - 일반 방식 대비 최대 400배 빠른 복구 성능

- **Immutable Backup (변경 불가 백업)**
  - 설정 기간 동안 백업본 삭제·변경 불가
  - 랜섬웨어 공격 등 관리자 계정 탈취 시에도 백업 보호

- **SureBackup (자동 복구 검증)**
  - 백업본의 복구 가능 여부를 자동으로 사전 검증
  - 실제 장애 상황에서 복구 신뢰도 보장

- **중복 제거·압축**
  - 소프트웨어 중복 제거로 스토리지 사용량 절감

***
## 3. 어디에 쓰이는가
- **가상화 환경 보호**
  - VMware vSphere, Microsoft Hyper-V VM 백업·복제·DR

- **클라우드 네이티브 백업**
  - AWS EC2, Azure VM, GCP 인스턴스 네이티브 백업 및 복구

- **SaaS 데이터 보호**
  - Microsoft 365(Exchange, SharePoint, Teams), Salesforce 데이터 백업

- **재해 복구(DR)**
  - CDP(Continuous Data Protection) 복제로 RPO/RTO 최소화
  - Disaster Recovery Orchestrator로 자동 장애 조치

- **물리 서버 통합 관리**
  - Windows, Linux 물리 서버도 동일 콘솔에서 백업 관리

***
## 4. 경쟁 솔루션과의 차이
| 항목 | Veeam | Commvault | Veritas NetBackup |
|---|---|---|---|
| **주요 강점** | 가상화·클라우드 특화 | 엔터프라이즈 통합 데이터 관리 | 대규모 물리 환경 강점 |
| **사용 편의성** | 직관적인 UI, 빠른 배포 | 기능 풍부하나 복잡 | 고성능이나 설정 복잡 |
| **클라우드 지원** | AWS·Azure·GCP 네이티브 | 지원하나 설정 복잡 | 지원하나 제한적 |
| **시장 점유율** | 글로벌 백업 시장 빅2, 포브스 500대 기업 83% 사용 | 중대형 기업 위주 | 전통적 대기업 위주 |

***
## 5. 한 줄 정리
Veeam은 **가상화·클라우드·SaaS를 단일 플랫폼으로 통합 보호**하는 세계 1위급 백업·복구 솔루션으로, 빠른 복구 속도와 변경 불가 백업으로 랜섬웨어 시대의 "데이터 복원력 표준"이다.

# Azure Resource Manager(ARM)란?

## 개념 정의
Azure Resource Manager(ARM)는 Microsoft Azure의 **클라우드 리소스의 생성·배포·구성·관리·삭제를 통합적으로 제어하는 관리 프레임워크(API 및 플랫폼)** 다. Azure의 VM, 네트워크, 스토리지, DB, 보안 등 모든 리소스의 라이프사이클을 코드/템플릿/명령어로 일관성 있게 관리할 수 있도록 설계됐다.

쉽게 말하면 “Azure 전체 리소스의 배포·정책·권한·감사·태그·자동화까지 모두 책임지는 통합 관제 시스템”이며, 클라우드 운영 자동화·IaC(Infrastructure as Code)의 기반이자 모든 Azure 관리의 중심이다.

## 핵심 특징

**자원 관리의 일관성**
- RESTful API/CLI/포털을 통한 모든 리소스 통합 관리
- 리소스를 리소스 그룹 내에 묶어 배포·정책·보안 관리 일관성 확보
- 변경·삭제·업데이트·감사 등 일괄 (원자적) 처리 지원

**ARM 템플릿 기반 자동화**
- JSON/YAML 형식의 ARM 템플릿으로 코드 기반 인프라 선언
- 템플릿에 따라 수십~수백 개 리소스를 한 번에 배포 및 업데이트
- 선언적 방식으로 의존성, 순서, 반복, 조건, 변수(파라미터) 모두 정의 가능

**정책·태그·권한 관리**
- 리소스(그룹) 단위로 태그 일괄 적용(비용·부서·환경 구분 등)
- RBAC(역할 기반 접근 제어) 및 정책(Policy) 적용 모든 작업 자동화
- 리소스별·그룹별 감사 로그 및 보안 이슈 한 번에 조회

**API 및 확장성**
- Azure REST API 및 SDK를 통한 모든 자원 자동화 제어
- Azure CLI, PowerShell, DevOps 파이프라인(IaC)과 연동 쉬움

**의존성 및 배포 검증**
- 복잡한 인프라 구조의 의존성·관계·구성을 코드 템플릿에 명확히 선언
- 배포 전 템플릿 유효성 검증, 오류 감지 및 롤백 기능

## 주요 활용 사례

- 대규모 서비스 환경을 ARM 템플릿 하나로 배포/업데이트/삭제
- DevOps 파이프라인에서 인프라(ID, 네트워크, VM, DB, 정책 등) 자동 생성 및 롤백
- Cost Center/부서·서비스별 자원 태그 배포 및 비용 분석 자동화
- 조직 정책 및 보안 기준 표준 템플릿화, 일관된 정책 자동 적용
- 운영환경 변환(Dev/QA/Prod 등) 시 전체 인프라 신속 복제·변경

## 기본 워크플로 및 관리 예시

### 1. ARM 템플릿 작성
- 인프라(리소스 그룹, VM, VNet, Storage 등) 선언적 JSON/YAML 코드 파일 정의

### 2. 리소스 배포(예시: CLI)
```bash
az deployment group create \
  --resource-group my-rg \
  --template-file azuredeploy.json \
  --parameters storageAccountName=mystorage
```

### 3. 정책 및 태그 일괄 적용
- Resource Manager API/CLI/포털에서 역할/정책/태그 그룹 단위로 적용

### 4. 배포/변경/삭제/업데이트 관리
- 템플릿 변경 시 코드로 인프라 전체 구조 자동 업데이트
- 배포 이력, 롤백, 오류 로그 실시간 기록/관리

### 5. DevOps 연동 및 IaC 자동화
- Azure DevOps, GitHub Action, Terraform 등 파이프라인에서 ARM 템플릿 기반 인프라 제어

## 비용 구조

- ARM 자체는 사용료 없음(리소스·API 호출만 과금)
- 템플릿·정책·자동화 기능 모두 무료, MS 지원 포함

## 보안 및 운영 모범 사례

- 선언적 인프라(IaC) 정규화 및 버전 관리(Git 등)
- ARM 템플릿에 보안·정책·태그·권한 자동 반영
- 리소스 그룹 단위로 RBAC·정책·감사 적용, Cost 관리 태그 사용
- 배포·삭제 이력 코드로 기록, 운영 신뢰성·보안 강화

Azure Resource Manager는 클라우드 인프라 관리의 표준이자 모든 Azure 운영·자동화의 기반으로, 신속·일관성·안정성·보안을 중심으로 조직의 인프라 환경을 효율적으로 관리하는 데 반드시 필요한 핵심 서비스다.

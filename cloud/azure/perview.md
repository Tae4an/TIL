# Azure Purview (Microsoft Purview)란?

## 개념 정의
Microsoft Purview는 클라우드 기반 데이터 거버넌스·카탈로그·보안 플랫폼입니다. 온프레미스, 멀티클라우드, SaaS 전반의 데이터를 자동으로 스캔, 분류, 계보(Lineage) 추적하며 중앙에서 데이터 정책과 접근을 관리할 수 있도록 돕습니다. 조직 전체 데이터가 어디에, 어떤 형식으로, 얼마나, 어떤 민감도로 있는지 한눈에 볼 수 있고, 규정에 맞게 통제·보호·공유할 수 있게 해줍니다.

## 핵심 기능

- **데이터 맵(Data Map)**: 여러 데이터 소스를 자동으로 스캔해 메타데이터를 수집하고 데이터 지형을 시각화하며 데이터 흐름 및 영향 분석에 활용할 수 있습니다.
- **데이터 카탈로그(Unified Catalog)**: 검색 가능한 카탈로그에 모든 데이터 자산을 저장하고 용어 사전, 설명, 소유자 정보 등 메타데이터를 관리해 데이터 과학자와 분석가들이 필요한 데이터를 쉽게 찾아 이해할 수 있습니다.
- **자동 분류 및 민감 데이터 보호**: 내장 및 사용자 정의 규칙으로 개인정보 등 민감 데이터를 자동 탐지하고 레이블링하며 일관된 분류 및 정책을 적용합니다.
- **정책 및 액세스 거버넌스(Data Policy)**: 중앙에서 데이터 접근, 마스킹, 보존, 삭제, 공유 정책을 정의하고 적용하여 데이터 수명주기 규칙을 자동화합니다.

## 장점 및 통합

- 온프레미스, Azure, 다른 클라우드, Microsoft 365, Power BI 등 다양한 데이터 소스를 단일 콘솔에서 관리할 수 있습니다.
- 데이터 위치를 변경하지 않고 메타데이터만 수집해 인덱싱하기 때문에 기존 시스템과 연동하여 거버넌스를 구축할 수 있습니다.
- 데이터 분류, 액세스 제어, 감사, 규정 준수 리포팅 기능을 지원해 GDPR 등 각종 규제 대응이 용이합니다.
- 역할·권한 모델 기반으로 최소 권한 원칙에 입각한 데이터 접근 통제가 가능합니다.
- 데이터 인사이트 대시보드를 통해 데이터 자산 분포, 민감 데이터 현황, 스캔 상태 등 거버넌스 상태를 실시간으로 파악할 수 있습니다.

## 활용 사례

- 도메인별 데이터 제품 관리 및 데이터 카탈로그 구축
- 민감 데이터 위치 파악, 레이블링, 정책 적용
- 멀티클라우드, 온프레미스 환경에서 통합 데이터 거버넌스 구현
- 분석 플랫폼과 연계해 신뢰할 수 있는 데이터 소스 관리

Microsoft Purview는 조직 데이터 자산을 효과적으로 찾아내고 이해하며 통제하는 데이터 거버넌스 솔루션으로, 데이터 기반 의사결정과 규정 준수를 지원하는 핵심 플랫폼입니다.

# Akamai Guardicore란

## 1. 개념 요약

Akamai Guardicore(가디코어)는 **데이터센터·클라우드·하이브리드 환경에서 에이전트 기반 마이크로세그멘테이션과 딥 가시성을 제공하는 제로 트러스트 네트워크 보안 솔루션**이다.  
Guardicore는 이스라엘 보안 스타트업으로 시작해 2021년 Akamai에 인수되었으며, **워크로드 간 트래픽을 세밀하게 제어하고 공격자의 횡방향 이동(Lateral Movement)을 차단하는 "데이터센터·클라우드 마이크로세그멘테이션 전문 솔루션"** 이라고 보면 된다.

앞서 정리한 Illumio Core와 유사한 포지션에 있으나, **위협 가시성·프로세스 수준 모니터링·위협 인텔리전스 연동** 면에서 보안 탐지 기능이 더 강조된다.

***

## 2. 특징

- **에이전트 기반 딥 가시성**
  - 각 워크로드에 에이전트를 설치해 프로세스 수준, 사용자 수준, 네트워크 연결 수준의 매우 상세한 트래픽 흐름을 수집한다.
  - "어떤 프로세스가 어떤 IP·포트로 통신하는지"까지 파악 가능해, 이상 징후 탐지에 유리하다.

- **레이블 기반 정책 모델**
  - IP·서브넷 기반이 아닌 애플리케이션·환경·역할 레이블을 기반으로 세그멘테이션 정책을 정의한다.
  - 클라우드 태그(AWS 태그, Azure 태그 등)와 자동 연동해 정책 매핑이 가능하다.

- **멀티 플랫폼 지원**
  - Windows, Linux(RHEL, Ubuntu 등), 컨테이너, 베어메탈, VMware, AWS, Azure, GCP 등 이기종 환경에서 동일한 정책을 적용할 수 있다.

- **위협 탐지 & 인텔리전스 연동**
  - 수집된 트래픽 데이터를 기반으로 의심스러운 통신 패턴을 탐지하고, 알림을 보낸다.
  - Akamai의 위협 인텔리전스 피드와 연동해 알려진 악성 IP·도메인 통신을 자동 탐지·차단 가능하다.

- **Reveal (가시성 맵)**
  - 전체 워크로드 간 트래픽을 실시간 시각화하는 그래픽 맵을 제공한다.
  - 새로운 환경에서 트래픽 흐름을 먼저 파악한 뒤 정책을 설계하는 워크플로를 지원한다.

- **자동화·오케스트레이션 연동**
  - REST API와 Terraform, Ansible, AWS CloudFormation, Kubernetes 등과 통합해 정책을 코드(Policy-as-Code)로 관리할 수 있다.

***

## 3. 어디에 쓰이는가

- **데이터센터 내부 횡방향 이동 차단**
  - 침해된 서버에서 인접 서버·DB로 확산하는 경로를 세그멘테이션 정책으로 차단.

- **랜섬웨어 확산 방지**
  - 감염 탐지 즉시 해당 워크로드를 격리하거나, 사전 세그멘테이션으로 확산 경로 자체를 제거.

- **핵심 인프라 보호**
  - Active Directory, DB 서버, 결제 시스템 등을 별도 세그먼트로 분리해 최소 권한 접근만 허용.

- **컴플라이언스 준수**
  - PCI DSS, HIPAA, NIST Zero Trust Architecture 등의 네트워크 세그멘테이션 요구사항을 구현·증명하는 감사 보고서 자동 생성.

- **클라우드 워크로드 보호**
  - AWS·Azure·GCP VM 및 컨테이너에 동일한 세그멘테이션 정책을 적용해 온프레미스와 일관된 보안 정책 유지.

***

## 4. Guardicore vs Illumio Core 비교

| 항목 | Akamai Guardicore | Illumio Core |
|---|---|---|
| 에이전트 | 에이전트 설치 필요 | 에이전트(VEN) 설치 필요 |
| 가시성 수준 | 프로세스·사용자 수준까지 상세 수집 | 프로세스·포트 수준 수집 |
| 위협 탐지 | 내장 이상 탐지 + 위협 인텔리전스 연동 | 탐지보다 정책 시행에 집중 |
| 정책 시행 방식 | OS 방화벽 직접 제어 | OS 방화벽 직접 제어 |
| 컨테이너 지원 | Kubernetes 네이티브 연동 강점 | Kubernetes 지원 |
| 클라우드 연동 | Akamai 플랫폼 생태계 통합 | Illumio CloudSecure 병용 |
| 공통점 | 레이블 기반 정책, 제로 트러스트 마이크로세그멘테이션, 횡방향 이동 차단 | |

***

## 5. 한 줄 정리

Akamai Guardicore는 **에이전트 기반 프로세스 수준 가시성과 마이크로세그멘테이션으로 데이터센터·클라우드 전반의 워크로드를 보호하며, 위협 탐지·인텔리전스 연동을 통해 공격자의 횡방향 이동을 탐지·차단하는 Akamai의 제로 트러스트 세그멘테이션 솔루션** 이다.

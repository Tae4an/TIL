# Zerto란

## 1. 개념 요약

Zerto는 **가상화·클라우드 환경에 특화된 재해복구(DR) 및 지속적 데이터 보호(CDP) 솔루션**이다.  
실시간에 가까운 데이터 복제와 자동화된 복구 오케스트레이션으로, 장애·랜섬웨어·재해 발생 시 수초 전 시점까지 서비스를 빠르게 복구하는 **IT 복원력(Resilience) 플랫폼**이라고 보면 된다.

***

## 2. 특징

- **CDP(Continuous Data Protection) 기반 실시간 복제**  
  - 전통적인 “1시간·1일 단위 백업”이 아니라, 변경 사항을 저널(journal)에 연속적으로 기록해 **수초 단위의 매우 낮은 RPO**를 제공한다.  
  - 저널에서 원하는 시점을 선택해 “몇 초 전” 상태로 되돌릴 수 있어 랜섬웨어 대응에 유리하다.

- **매우 낮은 RPO/RTO**  
  - RPO(복구 시 허용되는 데이터 손실)를 수초 내로 줄이고, RTO(서비스를 다시 올리는 데 걸리는 시간)도 수분 단위 수준까지 단축하도록 설계되어 있다.  

- **하이퍼바이저 기반 복제**  
  - 특정 스토리지 벤더에 종속되지 않고, VMware vSphere·Hyper-V 등 **하이퍼바이저 레벨에서 VM 단위로 복제**한다.  
  - 서로 다른 스토리지, 온프레미스–클라우드 간 이기종 환경에서도 일관된 DR 구성이 가능하다.

- **DR·백업·마이그레이션 통합**  
  - 재해복구, 단기·장기 보관(백업), 워크로드 마이그레이션(온프레미스↔클라우드, 클라우드↔클라우드)을 **하나의 플랫폼**에서 처리한다.  

- **자동화 오케스트레이션**  
  - Virtual Protection Group(VPG) 단위로 여러 VM을 하나의 애플리케이션으로 묶어, **Failover/Failback 시퀀스·부팅 순서·스크립트 실행** 등을 자동화할 수 있다.  

***

## 3. 어디에 쓰이는가

- **가상화 인프라 DR**  
  - VMware vSphere, Hyper-V 기반 데이터센터의 핵심 서비스를 DR 센터 또는 퍼블릭 클라우드(Azure, AWS, GCP 등)로 보호·복구하는 용도.  

- **랜섬웨어 대응**  
  - 감염 시점 이전 수초·수분 전 지점으로 빠르게 롤백해 암호화 피해를 최소화하고, 비즈니스 다운타임을 줄인다.  

- **클라우드 마이그레이션**  
  - 온프레미스 → 클라우드, 또는 클라우드 ↔ 클라우드 간 마이그레이션을 **서비스 중단을 최소화한 상태(near-zero downtime)** 로 수행할 때 사용된다.  

- **BCP/DR 모의훈련**  
  - 실제 서비스에 영향을 주지 않는 **비파괴 테스트 모드**로 DR 시나리오를 주기적으로 연습하고, 복구 시간·절차를 검증하는 데 활용된다.  

***

## 4. 주요 구성 요소

| 구성 요소 | 역할 |
|---|---|
| ZVM (Zerto Virtual Manager) | vCenter/Hyper-V와 연동되는 중앙 관리 서버, 정책·복제·Failover/Failback 오케스트레이션 총괄 |
| VRA (Virtual Replication Appliance) | 각 하이퍼바이저 호스트에 배포되는 가상 어플라이언스, VM I/O를 캡처해 타 사이트로 실시간 복제 |
| Journal | 변경 데이터가 시간 순서대로 쌓이는 영역, “특정 시점 복구(Point-in-time restore)”의 기반 |
| VPG (Virtual Protection Group) | 여러 VM을 하나의 애플리케이션 단위로 묶어 동일 시점으로 보호·복구하는 논리 그룹 |

***

## 5. 전통 백업 솔루션과의 차이

| 항목 | 전통 백업 | Zerto(CDP 기반) |
|---|---|---|
| 보호 방식 | 스케줄 기반 스냅샷(분·시간·일 단위) | 변경분 지속 캡처(CDP) |
| RPO | 분~시간 단위 | 수초 단위까지 가능 |
| RTO | 수십 분~수시간 | 수분 이내를 목표 |
| 주요 목적 | 장기 보관·법적 보존 | DR, 랜섬웨어 대응, 무중단 마이그레이션 |
| 보호 단위 | 파일·DB·이미지 등 혼합 | 주로 VM·애플리케이션 단위 |

***

## 6. 한 줄 정리

Zerto는 **CDP 기반의 실시간 복제와 자동 오케스트레이션으로, 가상화·클라우드 환경에서 재해복구·랜섬웨어 대응·마이그레이션을 통합 제공하는 하이브리드 클라우드 DR/IT 복원력 플랫폼**이다.

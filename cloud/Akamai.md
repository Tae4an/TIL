# Akamai란

## 1. 개념 요약

Akamai(아카마이)는 **세계 최대 규모의 분산 엣지 네트워크를 기반으로 CDN·웹 성능 최적화·사이버 보안·클라우드 컴퓨팅을 통합 제공하는 글로벌 플랫폼 기업**이다.  
1998년 MIT 출신들이 설립했으며, 전 세계 134개국 1,400개 이상의 엣지 네트워크를 통해 **전 세계 인터넷 트래픽의 약 35%를 처리**하는 인터넷 인프라의 핵심 플레이어다.  
초기에는 CDN 전문 기업으로 시작했지만, 현재는 **CDN + 보안 + 클라우드**를 아우르는 Akamai Connected Cloud 플랫폼으로 확장되었다.

***

## 2. 핵심 제품군

- **CDN(콘텐츠 전송 네트워크)**
  - 사용자와 가장 가까운 엣지 서버에서 웹·미디어·API 콘텐츠를 제공해 빠른 응답 속도와 안정성을 보장한다.
  - 대규모 트래픽 급증(세일, 스포츠 생중계 등)에도 오리진 서버 부하 없이 대응 가능하다.

- **웹·API 보안 (App & API Protector)**
  - WAF(웹 애플리케이션 방화벽), DDoS 방어, 봇 차단, API 보안을 단일 솔루션으로 통합 제공한다.
  - 엣지 레벨에서 공격 트래픽을 흡수·차단해 오리진 서버를 보호한다.

- **Zero Trust 보안 (Akamai Guardicore, EAA, SIA)**
  - Guardicore Segmentation: 데이터센터·클라우드 워크로드 간 마이크로세그멘테이션.
  - EAA(Enterprise Application Access): 에이전트 없는 Zero Trust 애플리케이션 접근(ZTNA).
  - SIA(Secure Internet Access): 클라우드 기반 보안 웹 게이트웨이(SWG).

- **엣지 컴퓨팅 & 클라우드 (Akamai Connected Cloud)**
  - Linode 인수를 통해 확장한 클라우드 컴퓨팅·엣지 서버리스 플랫폼.
  - AWS·Azure·GCP보다 엣지에 분산된 저렴한 클라우드 인프라를 제공한다.

- **DNS 보안 (Edge DNS)**
  - 대규모 엣지 인프라를 활용한 고가용성 권위 DNS 서비스.
  - DDoS 공격으로 DNS가 중단되는 상황을 방지한다.

***

## 3. 어디에 쓰이는가

- **대규모 미디어·스트리밍**
  - 넷플릭스, 유튜브 같은 대용량 동영상 스트리밍 콘텐츠를 전 세계에 빠르게 배포.

- **이커머스·금융 서비스**
  - 트래픽 급증에 안정적으로 대응하면서 웹 응답 속도를 높이고, WAF·봇 방어로 사기·스크래핑을 차단.

- **기업 보안 (Zero Trust)**
  - 임직원의 내부 앱 접근을 ZTNA로 보호하고, Guardicore로 데이터센터 내 워�로드 간 이동을 제어.

- **게임·소프트웨어 배포**
  - 전 세계 동시 업데이트·패치를 원본 서버 부하 없이 빠르게 배포.

- **공공·금융·의료 규제 환경**
  - 보안 정책 준수와 고가용성이 동시에 필요한 환경에서 CDN + 보안을 통합 제공.

## 4. 주요 경쟁사와 비교

| 항목 | Akamai | Cloudflare | AWS CloudFront |
|---|---|---|---|
| CDN 규모 | 세계 최대 수준, 엣지 1,400개+ | 300개+ 도시 PoP | AWS 리전·엣지 기반 |
| 보안 통합 | WAF·DDoS·ZTNA·세그멘테이션 전방위 | WAF·DDoS·Zero Trust | WAF·Shield 중심 |
| 클라우드 | Connected Cloud (Linode 기반) | Workers 서버리스 중심 | AWS 풀 스택 |
| 주요 강점 | 엔터프라이즈·미디어 대규모 환경 | 가성비·중소기업 친화 | AWS 생태계 통합 |
| 보안 전문성 | 업계 최고 수준의 위협 인텔리전스 | 빠른 성장세 | AWS 서비스 중심 |

***

## 5. 한 줄 정리

Akamai는 **전 세계 최대 규모의 엣지 네트워크를 기반으로 CDN·웹 성능·DDoS 방어·Zero Trust 보안·클라우드 컴퓨팅을 통합 제공하는 "인터넷 인프라와 보안의 글로벌 플랫폼"** 이다.

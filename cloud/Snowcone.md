# AWS Snowcone란?

## 개념 정의
AWS Snowcone은 AWS Snow Family 중 가장 작은 엣지 디바이스로, 네트워크가 불안정하거나 단절된 환경에서 데이터를 안전하게 수집·저장·처리한 뒤 AWS로 전송하기 위해 설계된 초소형 러기드 장비다.  
쉽게 말해 “들고 다닐 수 있는 작은 박스 하나에 스토리지랑 약간의 컴퓨팅을 넣어, 현장에서 데이터 모아서 암호화·처리하고 나중에 AWS로 옮기는 용도의 Edge + 데이터 이관 디바이스”다.

## 하드웨어·기본 스펙

- **폼 팩터·내구성**
  - 크기 약 9×6×3인치, 무게 약 2.1kg 수준으로 한 손에 들고 다닐 수 있다.
  - 충격·진동·온도 변화에 견디도록 설계된 러기드 케이스로, 공장·광산·야전·선박 같은 거친 환경을 전제로 한다.

- **스토리지 용량**
  - 약 8TB 사용 가능 스토리지(일반 모델 기준)로, 센서 데이터·로그·이미지 등 엣지 수집용에 적합하다.
  - SSD 기반 변형(Snowcone SSD)은 더 높은 내구성과 IOPS를 제공해 엣지 애플리케이션 실행에 최적화된다.

- **컴퓨팅 리소스**
  - 2 vCPU, 4GB RAM 수준의 컴퓨트가 탑재되어 있어, AWS IoT Greengrass, 작은 컨테이너 워크로드, 경량 분석·필터링 로직 등을 로컬에서 실행할 수 있다.
  - EC2 호환 AMI 기반으로 간단한 애플리케이션을 띄워, 데이터 전송 전에 전처리·집계·압축을 수행하는 패턴이 일반적이다.

- **전원·휴대성**
  - 기본은 AC 전원 어댑터로 구동하며, 선택적 배터리로 차량·야외 등 전원 인프라가 부족한 곳에서도 사용할 수 있다.

## 보안·데이터 전송 방식

- **보안·암호화**
  - 저장되는 모든 데이터는 256비트 키로 자동 암호화되며, 암호화 키는 AWS KMS에서 관리되고 디바이스에는 저장되지 않는다.
  - TPM 기반 하드웨어 Root of Trust, 변조 감지·변조 방지 설계로 분실·도난 시에도 데이터 노출을 최소화한다.

- **데이터 전송 옵션**
  - **오프라인 전송**  
    - 현장에서 Snowcone에 데이터를 채운 뒤 디바이스를 AWS로 물리 배송하면, AWS가 S3 등으로 데이터를 적재한다.  
    - 대역폭이 극도로 제한된 환경(선박, 광산, 군 통신망 등)에서 대용량 데이터를 빠르게 클라우드로 옮길 때 사용한다.
  - **온라인 전송 (DataSync)**  
    - Snowcone에 AWS DataSync 에이전트를 구동해, 가능한 네트워크 대역폭 범위 내에서 점진적으로 AWS로 동기화할 수 있다.  
    - 오프라인 배송과 병행하거나, “현장 캐시 + 백그라운드 전송” 패턴으로 많이 쓴다.

## 대표 활용 사례

- **Remote / Edge 환경 데이터 수집**
  - 인터넷 연결이 제한적인 공장, 광산, 유전, 선박, 원격 연구 기지 등에서 센서·로그·영상 데이터를 Snowcone에 수집·임시 저장 후, 물리 배송 또는 DataSync로 AWS에 업로드한다.
  - 현장에서 간단한 필터링·압축·익명화 작업을 실행해 전송량과 보안 리스크를 줄인다.

- **모바일·야전 시나리오**
  - 군/재난 대응, 차량·드론·배낭에 탑재하여 이동 중 데이터 수집 및 엣지 분석에 활용한다.
  - 실제로 Snowcone을 우주 궤도로 보내 위성에서 데이터를 처리·저장한 사례도 존재한다.

- **소규모 오프라인 마이그레이션**
  - 수~수십 TB 정도 온프레미스 데이터를 전용 회선 없이 비교적 저렴하게 AWS로 옮길 때 여러 대의 Snowcone을 사용한다.
  - 수백 TB~PB급이면 Snowball/Snowmobile이 더 적합하고, 그 이하·필드 작업이면 Snowcone이 적합한 포지션이다.

## Snow Family 내 위치

| 디바이스       | 용량(대략) | 주요 용도                            | 특징                                  |
|----------------|-----------|---------------------------------------|---------------------------------------|
| Snowcone       | ~8TB      | 엣지 수집·소형 마이그레이션          | 가장 작고 가벼운 러기드 장비, 소형 컴퓨트 탑재 |
| Snowball Edge  | 수십 TB   | 대규모 마이그레이션 + 강한 엣지 컴퓨팅 | 여러 vCPU·수백 GB RAM, GPU 옵션       |
| Snowmobile     | 최대 100PB| 초대규모 데이터센터 리프트 앤 시프트 | 트레일러 트럭 단위, 데이터센터 전체 이전 |

Snowcone은 “가방에 넣고 들고 다닐 수 있는 수준의 Snowball Edge”로 보면 되고, 주로 엣지 컴퓨팅과 제한된 네트워크 환경의 데이터 이관에 최적화된 디바이스다.

# Morpheus란?

## 1. 개념 요약

Morpheus(모피어스)는 **온프레미스·프라이빗 클라우드·퍼블릭 클라우드를 단일 콘솔에서 통합 관리하는 CMP(Cloud Management Platform)** 솔루션이다.
2015년 설립, 2024년 HPE에 인수되어 현재 **HPE Morpheus Enterprise Software**로 운영되며, **하이브리드·멀티 클라우드 환경의 프로비저닝·자동화·거버넌스·비용 관리를 올인원으로 처리하는 "클라우드 통합 운영 허브"** 다.

***

## 2. 특징

- **클라우드 애그노스틱(Agnostic)**
  - AWS·Azure·GCP·VMware·Hyper-V·KVM·OpenStack 등 90개 이상 인프라 통합 지원
  - 특정 벤더에 종속되지 않는 멀티 클라우드 운영 가능

- **셀프 서비스 포털**
  - 개발자·운영팀이 승인 워크플로 안에서 직접 VM·컨테이너·앱 프로비저닝
  - IT 거버넌스 정책(가드레일) 안에서 자율적으로 배포 가능

- **통합 자동화·오케스트레이션**
  - 프로비저닝부터 라이프사이클 관리·폐기까지 자동화
  - Terraform·Ansible·CI/CD 파이프라인 네이티브 연동

- **비용 관리(FinOps)**
  - 온프레미스·퍼블릭 클라우드 비용을 중앙에서 통합 집계·분석
  - 예산 정책, 알림, 적정 규모 조정 제안

- **내장 Kubernetes 지원**
  - CNCF 인증 Morpheus Kubernetes Service(MKS)로 K8s 배포·관리

***

## 3. 어디에 쓰이는가

- **멀티 클라우드 통합 운영**
  - AWS·Azure·GCP를 하나의 콘솔에서 동시 관리, 클라우드 간 워크로드 이동

- **프라이빗 클라우드 구축**
  - VMware 대체 또는 보완으로 온프레미스 셀프서비스 클라우드 구현

- **IT 서비스 자동화**
  - ITSM(ServiceNow 등)과 연동해 클라우드 자원 요청·승인·프로비저닝 자동화

- **거버넌스·컴플라이언스**
  - 역할 기반 접근 제어(RBAC), 정책 자동 시행으로 보안 규정 준수

- **비용 최적화(FinOps)**
  - 사용량·청구 데이터 통합 분석으로 불필요한 클라우드 비용 절감

***

## 4. 유사 솔루션과의 비교

| 항목 | Morpheus | VMware vRealize | AWS Control Tower |
|---|---|---|---|
| **멀티 클라우드** | AWS·Azure·GCP·온프레미스 모두 | VMware 중심 | AWS 단일 클라우드 |
| **벤더 종속** | 없음 (애그노스틱) | VMware 종속 | AWS 종속 |
| **자동화** | 내장 오케스트레이션 | vROps 연동 필요 | 제한적 |
| **K8s 지원** | MKS 내장 | Tanzu 별도 | EKS 별도 |
| **주요 고객** | 엔터프라이즈·공공 | VMware 기존 고객 | AWS 올인 기업 |

***

## 5. 한 줄 정리

Morpheus는 **온프레미스부터 멀티 퍼블릭 클라우드까지 단일 콘솔로 프로비저닝·자동화·비용 관리·거버넌스를 통합 처리**하는 HPE의 "하이브리드 클라우드 올인원 관리 플랫폼"이다.

# Horizon이란?

## 1. 개념 요약

Horizon(보통 VMware Horizon, 최근 명칭은 Omnissa Horizon)은 **데스크톱·애플리케이션 가상화를 제공하는 VDI(가상 데스크톱 인프라) 솔루션**이다.  
즉, 사용자 PC 환경(Windows/Linux 데스크톱과 업무 앱)을 데이터센터·클라우드에서 가상 머신으로 실행시키고, 사용자는 어디서나 원격 접속해 사용하는 플랫폼이다.

***

## 2. 특징

- **가상 데스크톱 & 애플리케이션 제공**  
  - Windows·Linux 가상 데스크톱, RDS 기반 애플리케이션, 물리 PC 원격 접속까지 단일 플랫폼에서 제공한다.  

- **중앙 집중 관리**  
  - Horizon Connection Server에서 사용자 인증, 데스크톱 풀 할당, 세션 연결을 중앙에서 관리한다.  

- **VMware 인프라와 긴밀한 통합**  
  - vSphere(가상화), vSAN(스토리지), NSX(네트워크 가상화)와 연동해 전체 VMware 스택 기반 VDI 구성이 가능하다.  

- **하이브리드·멀티 클라우드 지원**  
  - 온프레미스뿐 아니라 VMware Cloud on AWS, Azure VMware Solution 등 다양한 클라우드 환경에 배포할 수 있다.  

***

## 3. 어디에 쓰이는가

- **사내 업무용 VDI 환경**  
  - 콜센터, 금융, 공공기관 등에서 내부망 전용 가상 데스크톱을 제공해 데이터 유출을 줄인다.  

- **재택·원격근무 지원**  
  - 임직원이 집·지사·해외에서 사내 VDI에 접속해 동일한 업무 환경으로 일할 수 있게 한다.  

- **보안이 중요한 환경**  
  - 실제 데이터는 데이터센터에만 두고, 단말에는 화면만 전송해 단말 분실·악성코드에 의한 정보 유출 위험을 완화한다.  

- **표준 데스크톱 이미지 운영**  
  - 골든 이미지 기반으로 수백·수천 명에게 동일한 데스크톱을 빠르게 배포하고, 패치·업데이트를 중앙에서 일괄 적용한다.  

***

## 4. 주요 구성 요소

| 구성 요소 | 역할 |
|---|---|
| Horizon Connection Server | 사용자 인증, 데스크톱/애플리케이션 풀 매핑, 세션 브로커 |
| vSphere/ESXi | 가상 데스크톱 VM을 실행하는 하이퍼바이저 계층 |
| vCenter Server | VM·호스트·템플릿·스냅샷 관리 |
| Horizon Agent | VDI VM 내부에 설치되어 세션, 주변기기, 클립보드 등 기능 제공 |
| Horizon Client / HTML5 | 사용자가 PC·노트북·Thin Client·브라우저로 접속할 때 사용하는 클라이언트 |

***

## 5. 한 줄 정리

Horizon은 **사용자 데스크톱과 애플리케이션을 데이터센터·클라우드에서 가상 머신으로 실행하고, 어디서나 안전하게 접속해 사용하게 해주는 엔터프라이즈용 VDI·앱 가상화 플랫폼**이다.

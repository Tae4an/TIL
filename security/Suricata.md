# Suricata란?

## 1. 개념 요약

Suricata는 **OISF에서 개발한 오픈소스 네트워크 보안 엔진**으로, **IDS(침입탐지), IPS(침입방지), NSM(네트워크보안모니터링)** 을 모두 지원한다.  
**멀티스레딩과 고속 패킷 처리로 Snort 대안으로 개발된 "고성능 L7 애플리케이션 계층 분석 도구"** 다.

***

## 2. 특징

- **다중 모드 지원**  
  - IDS(패킷 미러링 모니터링), IPS(인라인 차단), NSM(상세 로깅)  

- **멀티스레딩 아키텍처**  
  - 다중 CPU 코어 활용 → 10Gbps+ 고속 처리 가능  

- **L7 프로토콜 분석**  
  - HTTP/2, TLS, 파일 추출 등 애플리케이션 계층 깊이 검사  

- **Snort 룰 호환**  
  - Emerging Threats 등 Snort 룰셋 그대로 사용 가능  

- **풍부한 출력 형식**  
  - JSON(EVE), Lua 스크립팅, 파일 추출 등 분석 친화적  

***

## 3. 어디에 쓰이는가

- **네트워크 침입 방어**  
  - pfSense, Security Onion 등 플랫폼 패키지로 IDS/IPS 구축  

- **보안 모니터링**  
  - ELK 스택과 연동해 네트워크 트래픽 시각화·분석  

- **클라우드 네트워크 보호**  
  - AWS/GCP VPC 트래픽 모니터링, 인라인 IPS  

- **취약점 탐지**  
  - 알려진 익스플로잇, C2 통신, 악성 페이로드 탐지  

***

## 4. Snort와의 차이

| 항목 | Snort | Suricata |
|------|-------|----------|
| **스레딩** | 싱글스레드 | 멀티스레딩(고속 처리) |
| **프로토콜 분석** | 기본 | L7 심층 검사(TLS, HTTP/2) |
| **성능** | 1-2Gbps 제한 | 10Gbps+ 지원 |
| **출력 형식** | 텍스트/통계 | JSON, Lua, 파일 추출 |
| **개발 상태** | 성숙 | 활발한 신기능 추가 |

Suricata가 **현대 고속 네트워크 환경에 더 적합**하다.

***

## 5. 한 줄 정리

Suricata는 **오픈소스 멀티스레딩 IDS/IPS/NSM 엔진**으로, L7 애플리케이션 분석과 Snort 룰 호환성을 갖춰 고성능 네트워크 보안 모니터링의 "현대 표준"이다.
